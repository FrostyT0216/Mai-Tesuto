## 📖 什么是 Mai-Tesuto？

**Mai-Tesuto** 是一个运行在你电脑上的**本地刷题练习工具**。它不需要联网，不需要安装复杂的软件，只要你的电脑有 **Node.js**，双击一个文件就能开始刷题。

无论你是在准备考试、复习知识点、还是想通过反复练习来巩固学习内容，Mai-Tesuto 都能帮助你：

- 📋 **多题库管理** — 可以同时管理多个题库，随时切换
- 🎯 **顺序 / 随机练习** — 想按章节顺序来还是随机打乱，随你选
- 📊 **正确率统计** — 每道题做了多少次、对了几次、错了几次，一目了然
- 📒 **错题本** — 自动整理你做错的题目，方便针对性复习
- 💾 **数据本地保存** — 所有学习记录都存在你自己的电脑上，安全可靠
- 🎨 **清新界面** — 采用 Material Design 风格，看着舒服，用着顺手

> **Mai-Tesuto** 这个名字来源于日语「毎（まい）」+「テスト」，意为"每日一测"，希望你能每天坚持练习，积少成多。

## 🚀 快速开始

### 前置条件（只需要做一次）

你的电脑需要安装 **Node.js**。如果你还没装过：

1. 打开浏览器，访问 <https://nodejs.org>
2. 下载 **LTS 版本**（长期支持版，推荐大多数用户使用）
3. 双击下载的安装包，一路点击"下一步"完成安装

> 💡 **怎么确认 Node.js 装好了？** 按 `Win + R`，输入 `cmd` 回车，在弹出的黑色窗口中输入 `node -v`，如果看到类似 `v18.x.x` 的版本号，就说明安装成功了。

### 启动程序

1. 下载本项目到你电脑的任意文件夹
2. 双击 `start.bat` 文件
3. 等待 2 秒，浏览器会自动打开刷题页面

就是这么简单！🎉

> ⚠️ **注意**：如果双击 `start.bat` 没反应，可以右键点击 `start.bat`，选择"以管理员身份运行"试试。

## 📚 详细使用指南

### 1. 启动页面 — 配置你的练习

打开程序后，你会看到**启动配置页**，这里有三个设置项：

**A. 练习模式**

- **顺序模式**：按照题库中题目的原始顺序出题
- **随机模式**：把所有题目打乱后随机出题

**B. 题目数量**

- 在输入框里填写你想练习的题目数量（比如 20）
- 或者打开"全部题目"开关，一次性练习题库中的所有题目

**C. 章节筛选**（可选）

- 如果题库有多个章节，你可以选择只练习某个或某几个章节
- 不选的话默认练习所有章节

设置完毕后，点击底部的 **"开始练习"** 按钮，进入答题界面。

***

### 2. 答题页面 — 开始刷题

进入答题页后，你会看到：

- **顶部进度条**：显示当前进度（比如 3/20，表示 20 题中正在做第 3 题）
- **题目卡片**：显示题干和所有选项
- **选项区域**：
  - 单选题：圆形的单选按钮，点击即可选中
  - 多选题：方形的多选框，可以勾选多个答案
  - 判断题：只有"正确"和"错误"两个选项

**操作流程**：

1. 阅读题目，选择你认为正确的答案
2. 点击 **"提交答案"** 按钮
3. 系统会立刻告诉你答对了还是答错了，并显示正确答案和解析（如果有的话）
4. 点击 **"下一题"** 继续
5. 你也可以随时点击题号跳转到任意题目

> 💡 答错的题目会自动记录到错题本，方便你之后复习。

完成所有题目后，自动跳转到统计看板页。

***

### 3. 统计看板页面 — 查看你的学习数据

做完题目后（或随时点击底部的统计按钮），你会看到统计看板：

**正确率统计**

- 按正确率从高到低排列所有题目
- 可以看到每道题：总作答次数、正确次数、错误次数、正确率百分比
- 点击排序按钮可以切换从高到低 / 从低到高排列

**错题本**

- 只显示你至少答错过一次的题目
- 按错误次数降序排列，错得最多的题目排在最前面
- 红色标记让你一眼就能注意到薄弱环节

**练习历史**

- 记录每次练习的概况：时间、模式、题目数量、正确率

***

### 4. 题库管理

Mai-Tesuto 支持多题库，你可以在首页切换题库：

- **切换题库**：在题库下拉菜单中选择，立即生效
- **创建新题库**：点击新建按钮，输入题库名称和题目数据
- **重命名题库**：点击题库旁边的编辑按钮
- **删除题库**：点击删除按钮（注意：删除后无法恢复！）

***

### 5. 数据导出与导入

**导出数据**：在统计页面点击"导出数据"，浏览器会自动下载一个 JSON 文件，里面包含了你所有的学习统计数据。

**导入数据**：在统计页面点击"导入数据"，选择之前导出的 JSON 文件，你的统计记录就会恢复。

> 💡 建议定期导出数据备份，以防误删。

***

## 📝 题库格式说明

题库是 JSON 格式的文件，结构如下：

```json
[
  {
    "chapter": "第一章：基础知识",
    "questions": [
      {
        "global_id": "Q001",
        "id": 1,
        "type": "单选题",
        "question": "以下哪个是中国的首都？",
        "options": {
          "A": "上海",
          "B": "北京",
          "C": "广州",
          "D": "深圳"
        },
        "answer": "B"
      }
    ]
  },
  {
    "chapter": "第二章：进阶内容",
    "questions": [
      {
        "global_id": "Q010",
        "id": 10,
        "type": "多选题",
        "question": "以下哪些是哺乳动物？（多选）",
        "options": {
          "A": "鲸鱼",
          "B": "鲨鱼",
          "C": "蝙蝠",
          "D": "企鹅"
        },
        "answer": "AC"
      },
      {
        "global_id": "Q011",
        "id": 11,
        "type": "判断题",
        "question": "地球是太阳系中最大的行星。",
        "answer": "错"
      }
    ]
  }
]
```

### 字段说明

| 字段          | 类型  | 必填 | 说明                                                  |
| ----------- | --- | -- | --------------------------------------------------- |
| `chapter`   | 字符串 | ✅  | 章节名称                                                |
| `questions` | 数组  | ✅  | 该章节下的题目列表                                           |
| `global_id` | 字符串 | ✅  | 全局唯一 ID，用于追踪统计                                      |
| `id`        | 数字  | ✅  | 章节内题号                                               |
| `type`      | 字符串 | ✅  | 题型：`单选题`、`多选题`、`判断题`                                |
| `question`  | 字符串 | ✅  | 题目内容                                                |
| `options`   | 对象  | ❌  | 选项（单选题/多选题必填，判断题不需要）                                |
| `answer`    | 字符串 | ✅  | 正确答案。单选题填字母（如 `B`），多选题填字母组合（如 `ABC`），判断题填 `对` 或 `错` |

### 如何添加自己的题库？

#### 方式一：手动编写 JSON（适合少量题目）

1. 在 `questions/` 文件夹下创建一个新的 `.json` 文件
2. 按照上面的格式填写你的题目
3. 在程序中通过"创建新题库"功能导入

#### 方式二：从学习通等平台的 PDF 题库自动生成（推荐！）

如果你在学习通、雨课堂等平台上有课后练习或作业题库，可以用以下方法快速转成 Mai-Tesuto 能用的 JSON 题库：

**第 1 步：导出 PDF**

以 **学习通** 为例：

1. 在电脑浏览器中打开学习通网页版，进入你要导出题库的章节/作业页面
2. 确保页面上显示了所有题目（如果有分页，需要逐页展开或加载完所有题目）
3. 按 `Ctrl + P`（或右键选择"打印"），调出浏览器的打印对话框
4. 在打印设置中，将"目标打印机"选择为 **"另存为 PDF"**
5. 点击"保存"，将 PDF 文件保存到电脑上

> 💡 **其他平台同理**：只要是网页上能看到的题目，都可以用浏览器的"打印 → 另存为 PDF"功能导出。

**第 2 步：用 AI 生成 JSON 题库**

将上一步保存的 PDF 文件发送给 AI（如 ChatGPT、Claude、Kimi 等），并附上以下提示词：

```
请将这份 PDF 中的题目转换成以下 JSON 格式，用于刷题程序：

格式要求：
- 最外层是一个数组
- 每个元素是一个章节对象，包含 "chapter"（章节名称）和 "questions"（题目数组）
- 每道题包含：global_id（全局唯一ID，如 Q001）、id（章节内题号）、type（单选题/多选题/判断题）、question（题目内容）、options（选项对象，判断题不需要）、answer（正确答案）
- 单选题 answer 填一个字母，如 "B"
- 多选题 answer 填多个字母，如 "AC"
- 判断题 answer 填 "对" 或 "错"

示例：
[
  {
    "chapter": "第一章",
    "questions": [
      {
        "global_id": "Q001",
        "id": 1,
        "type": "单选题",
        "question": "中国的首都是？",
        "options": { "A": "上海", "B": "北京", "C": "广州", "D": "深圳" },
        "answer": "B"
      }
    ]
  }
]

请尽量保留题目的原始内容，不要修改答案。如果 PDF 中有解析，也请一并保留在 "analysis" 字段中。
```

**第 3 步：保存并导入**

1. 将 AI 生成的 JSON 内容复制到一个新的 `.json` 文件中
2. 把文件放到 `questions/` 文件夹下
3. 启动 Mai-Tesuto，在程序首页的题库管理中选择"导入题库"或"创建新题库"
4. 给题库起个名字，粘贴 JSON 内容，完成导入

> ⚠️ **小贴士**：
>
> - AI 生成的 JSON 建议先检查一下格式是否正确，可以用在线 JSON 校验工具（如 json.cn）验证
> - 如果题目很多，AI 可能一次生成不完，可以分章节多次请求
> - 如果 PDF 是图片格式（文字无法选中），需要先使用 OCR 工具（如 Adobe Acrobat、WPS、或在线 OCR）提取文字后再交给 AI

### API 接口文档

程序运行后，所有 API 都在 `http://localhost:8080/api/` 下：

#### 题库相关

| 方法       | 路径                    | 说明                 |
| -------- | --------------------- | ------------------ |
| `GET`    | `/api/questions`      | 获取当前激活题库的所有题目      |
| `POST`   | `/api/questions`      | 替换当前激活题库的题目        |
| `GET`    | `/api/banks`          | 获取所有题库列表（含题目数、章节数） |
| `GET`    | `/api/banks?name=xxx` | 获取指定题库的题目          |
| `POST`   | `/api/banks`          | 创建新题库              |
| `PUT`    | `/api/banks`          | 更新现有题库             |
| `DELETE` | `/api/banks`          | 删除题库               |
| `POST`   | `/api/banks/active`   | 设置当前激活的题库          |
| `POST`   | `/api/banks/rename`   | 重命名题库              |

#### 统计相关

| 方法     | 路径           | 说明       |
| ------ | ------------ | -------- |
| `GET`  | `/api/stats` | 获取学习统计数据 |
| `POST` | `/api/stats` | 保存学习统计数据 |

### 如何自己修改或扩展

1. **修改界面样式**：所有 CSS 都在 `index.html` 的 `<style>` 标签内，使用 CSS 变量管理主题色，修改 `:root` 下的变量即可快速换肤
2. **添加新功能**：JavaScript 逻辑也在 `index.html` 中，采用模块化设计，各功能模块职责清晰
3. **修改服务器**：`server.js` 是纯 Node.js 原生代码，无需安装任何依赖，改完直接重启即可

### 启动方式

```bash
# 方式一：双击启动（推荐）
双击 start.bat

# 方式二：命令行启动
node server.js
# 然后在浏览器打开 http://localhost:8080
```

## ❓ 常见问题

<details>
<summary><strong>Q: 为什么双击 start.bat 后浏览器显示"无法访问此网站"？</strong></summary>

可能是 8080 端口被其他程序占用了。可以尝试：

1. 关闭可能占用 8080 端口的程序（如其他本地服务器、某些开发工具）
2. 或者修改 `server.js` 第 6 行的 `const PORT = 8080;` 换一个端口号（如 3000）

</details>

<details>
<summary><strong>Q: 我的数据存在哪里？安全吗？</strong></summary>

你的学习统计数据存在两个地方：

- `userdata/stats.json`（服务端文件）
- 浏览器 localStorage（客户端备份）

所有数据都在你本地电脑的 `userdata/` 文件夹中，不会上传到任何地方，非常安全。

</details>

<details>
<summary><strong>Q: 我换了电脑，怎么把数据迁移过去？</strong></summary>

1. 在旧电脑上，打开程序进入统计页面，点击"导出数据"，得到一个 JSON 文件
2. 把整个项目文件夹和导出的 JSON 文件复制到新电脑
3. 在新电脑上打开程序，进入统计页面，点击"导入数据"，选择那个 JSON 文件

或者直接把旧电脑上的 `userdata/` 文件夹复制到新电脑的项目目录下也可以。

</details>

<details>
<summary><strong>Q: 我能在手机上用吗？</strong></summary>

Mai-Tesuto 是为电脑设计的本地程序，手机无法直接运行。但它是响应式设计的，如果你在电脑上运行后，同一局域网下的手机可以通过电脑 IP 访问（如 `http://192.168.x.x:8080`）。

</details>

<details>
<summary><strong>Q: 程序会收集我的数据吗？</strong></summary>

不会。Mai-Tesuto 是纯本地运行的程序，没有任何网络请求发送到外部。你的所有数据都在你自己的电脑上。

</details>

<details>
<summary><strong>Q: 我有很多题目，会不会卡？</strong></summary>

Mai-Tesuto 采用轻量级设计，即使有几千道题目也能流畅运行。数据存储使用 JSON 文件格式，读写效率很高。

</details>
